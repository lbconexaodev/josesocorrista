# José Socorrista

Site da pré-campanha de José Socorrista em Boa Vista.

## Repositório

GitHub onde o projeto deve ser guardado:

```text
https://github.com/lbconexaodev/josesocorrista.git
```

## Rodar localmente

**Pré-requisito:** Node.js

1. Instale as dependências:

```bash
npm install
```

2. Configure o `.env` com a URL do Google Apps Script:

```env
VITE_GOOGLE_SHEETS_WEB_APP_URL="https://script.google.com/macros/s/SEU_ID/exec"
```

3. Rode o projeto:

```bash
npm run dev
```
